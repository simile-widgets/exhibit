About and Sources

About

The week facet was developed as part of MIT's Exhibit 3.0 framework. It allows you to display weekly events on a calendar as well as show details about these events with the rest of Exhibit's functionality.

How to Use

To use the week facet, include this facet using 

data-ex-facet-class="Week"
You JSON file should include the following fields (in addition to the fields required by Exhibit):
  1. startDate: Date in which your activity starts. If it is a recurring event, then all dates within the startDate/endDate range may display the event. If your event is not recurring, make the startDate and endDate the same.
  2. endDate: The ending date for your event.
  3. startTime: The time your event starts.
  4. endTime: The time your event ends.
  5. display: Whether the calendar will display your event. Enter either "true" or "false".
  6. label: The label for your event. This will be displayed on the calendar within each box for each event.
  7. description: Your event description.
  8. recurring: If your event is recurring, list the days of the week it occurs on. Use "M", "T", "W", "R", "F", "S", and "Sn" for "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", and "Sunday", respectively.

You can set the following settings for your Week Facet calendar:

  1. mini: "true" or "false": whether you want the mini version of the calendar.
  2. daystart: The first hour of the day that you want to display. For example, if you want your calendar to start at 1:00PM, write "1".
  3. dayend: The last hour of the day you want to display.
  4. startdate: The first day you want your calendar to display. For example, if you want your calendar to display a week starting with "07/20/2014" enter this into the field.
  5. enddate: The last day you want your calendar to display. If you specify both "startDate" and "endDate", "startDate" takes precendence.
  6. gridwidth: The width of your calendar.
  7. gridheight: The height of your calendar.
  8. x-cell-width: The horizontal width of each cell in the grid.
  9. y-cell-width: The height of each cell in the grid.
