Exhibit 3.0 Scripted Demo: Big Data

This demo lets you test the performance of (changes to) exhibit on an artificial large data set. File slow.html links to version 3.0.0 of exhibit which has a slow loader; fast.html links to the current version. Both load from test.js , which by default contains 25000 items; you can copy varying data sets onto test.js to test them. Perl file makebig.pl can generate other data set sizes.
